### STEPS:

Clone the repository


### STEP 01- Create a conda environment after opening the repository

```bash
conda create --prefix ./venv python=3.10 -y

```

```bash
conda activate ./venv
```

### STEP 02- install the requirements
```bash
pip install -r requirements.txt
```
